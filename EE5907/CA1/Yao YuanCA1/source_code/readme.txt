## About
Four files end with '.ipynb' are the source code for 4 parts of CA1.

## Requirements
You may need to download jupyter-notebook first, and open files with Jupyter console.
Please note that you need add source dataset 'spamData.mat' to the same folder with '.ipynb' files.
The codes are written in Python 3.7.6 and main packages used are 'scipy', 'pandas', 'numpy' and 'matplotlib'.
