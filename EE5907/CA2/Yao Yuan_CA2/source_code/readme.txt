## About
Five files end with '.ipynb' are the source code for 5 parts of CA2. 
model.png is the CNN structure for the part 5.

## Requirements
You may need to download jupyter-notebook first, and open files with Jupyter console.
Please note that you need add source dataset 'PIE' to the folder 'source_code'.
The codes are written in Python 3.7.6 and main packages used are 'tensorflow', 'keras', 'graphviz' , 'sklearn', 'numpy' and 'matplotlib'.
